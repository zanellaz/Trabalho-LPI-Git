Git

O que é git?

Git é um sistema de controle de versão distribuído gratuitamente e de código aberto projetado para lidar com tudo, desde projetos pequenos a muito grandes com velocidade e eficiência.

O Git é fácil de aprender e tem um desempenho extremamente rápido. Ele supera ferramentas SCM como Subversion, CVS, Perforce e ClearCase com recursos como ramificação local barata, áreas de teste convenientes e vários fluxos de trabalho.

Para navegar pelo site, basta usar a aba superior, ela indica todas as páginas para você acessar. Nas páginas, ao lado esquerdo, terá um sumário para você navegar pela página.

Neste site, você irá aprender sobre git, desde o básico, sua história ou até alguns tutoriais.



Espero que você goste!
