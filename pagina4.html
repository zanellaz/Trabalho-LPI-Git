Usando o Git

Comandos básicos

Para fazer um bom uso do Git, é necessário saber alguns comandos básicos, aqui vai uma lista dos principais (e suas funções).


git init

Inicializa seu repositório Git


git add [file]

Adiciona arquivos que você quer incluir atualizações no próximo commit.


git commit -m "message"

Cria um ponto na história com a mensagem que você põe.


git log

Mostra o seu commit com as suas mudanças.


git show [commit]

Mostra os commits que você fez
