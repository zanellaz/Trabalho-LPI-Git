Pro Git Book

Para explicar todas funcionalidades e uma pequena parte da história por trás desse software, Scott Chacon, fundador do GitHub, e Ben Straub escreveram um livro sobre tudo isso e muito mais, que foi publicado pela Apress.

Este livro conta com muitas informações interessantes para interessados em todo conteúdo envolvendo Git e seus derivados, tendo acesso gratuito para sua versão digital e completa.

O livro conta a origem desse software e sua história por trás de tudo. Mas ensina principalmente muitos variados comandos e métodos que podem ser usados pelo Git e seus derivados, e também aprofunda cada fundamento que o Git fornece para os desenvolvedores.

Fornece desde o básico até o avançado de cada utensílio que pode ser utilizado no desenvolvimento de qualquer projeto que venha precisar de uma ferramenta Git, mostrando desde imagens para explicação, até instruções e linhas de código para entendimento dos comandos, como este seguinte trecho demonstrando como obter o diretório de um repositório Git:

Inicializando um Repositório em um Diretório Existente
Para você começar a monitorar um projeto existente com Git, você deve ir para o diretório desse projeto.
Se você nunca fez isso, use o comando a seguir, que terá uma pequena diferença dependendo do sistema em que está executando:
para Linux:
$ cd /home/user/your_repository
para Mac:
$ cd /Users/user/your_repository
para Windows:
$ cd /c/user/your_repository
depois digite:
$ git init
